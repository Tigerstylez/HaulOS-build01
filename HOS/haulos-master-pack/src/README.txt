HaulOS Master Pack & Prototype

HaulOS is the real-time operating system for heavy transport. This repository provides the high-fidelity interactive prototype and investor pitch deck.

Features

Investor Deck: Interactive walkthrough of problem, solution, and roadmap.

Clickable Prototype: - Vehicle Profile Setup (Constraint-Aware logic).

Map View with Fuel intelligence.

One-tap status reporting.

Digital Logbook simulation.

Tech Stack

Framework: React 18 + Vite

Styling: Tailwind CSS

Icons: Lucide-React

Getting Started

Clone the repo

Install dependencies: npm install

Run Dev Server: npm run dev

Deployment

This project is ready to be deployed to Vercel or GitHub Pages.
